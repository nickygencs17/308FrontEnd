Our Project

Gerrymanering:

Gerrymandering is a practice intended to establish a political advantage for a particular political party or group by manipulating the district or constituency boundaries. Gerrymandering is effective because of the wasted vote effect i.e. votes are votes that did not contribute to electing a candidate.

Objective:

The aim of this project was to implement a software that would redistrict the boundaries of the congressional districts (each represented in the United States House of Representatives) and display more compact and politicially fair districts. Further details regarding this can be found on the course web page. (http://www3.cs.stonybrook.edu/~cse308/Assignments/DistrictGeneration.html)

Learning Outcomes:

Learning Outcomes: The course gave us an insight into Professional Software Development, and emphasizes the development of reliable and maintainable software via system requirements and specifications, software design methodologies including object-oriented design, implementation, integration, and testing; and software project management.

Technologies:

  1. Java Programming Langauge.
  2. Spring framework.
  3. Angular.
  4. MySQL databases.
